GeoJSON Storage, Editing, and Visualization RESTful GeoJSON Datastore

This demo provides a RESTful front-end for a GeoJSON data store, which plugs in to ElasticSearch, InfluxDB, MongoDB, or Redis back ends. The HTML5 uses WebGL to display live, interactive mapping visualizations and layer editors.

View Live Data Feeds

Real-time GeoJSON data displayed on a WebGL-enabled Mapbox map. Data store can be connected to ElasticSearch, InfluxDB, MongoDB, or Redis.

Edit GeoJSON in the Browser

GeoJSON may be edited directly from your web browser. Mapbox's Open Source GeoJSON.io editor has been modified to save data to the database.

RESTful GeoJSON Datastore

Clients may create, retreive, update, and delete (CRUD) GeoJSON objects to the datastore using a RESTful interface.

Rapid Development

Using Node.js packages Consolidate and Restify, the entire framework becomes a lightweight layer on top of ElasticSearch and other pluggable back ends.
